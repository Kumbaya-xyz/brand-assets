```

│   └── black-inverse/
```

## File Formats

- **SVG**: Vector format, scalable to any size. Ideal for web and print.
- **PNG @1x**: Standard resolution (1x scale)
- **PNG @2x**: Retina/HiDPI resolution (2x scale)
- **PNG @3x**: Super Retina resolution (3x scale)

## Usage Guidelines

When using Kumbaya brand assets:
- Use SVG files when possible for best quality at any size
- Use the appropriate PNG resolution for your target display density
- Maintain aspect ratios - do not stretch or distort logos
- Ensure adequate contrast with backgrounds
